### 🤖 BASIC ROBOTICS SIMULATIONS (NO HARDWARE)

*   [ ] **Virtual Turtle Maze Solver**
    *   Uses Python's built-in `turtle` graphics library.
    *   Navigates a digital "robot" out of a custom wall maze autonomously.
*   [ ] **Inverse Kinematics Calculator**
    *   Calculates necessary joint angles for robotic arms.
    *   Takes user $(X, Y)$ coordinates and outputs exact motor angles.
*   [ ] **Virtual Joystick Rover Controller**
    *   Builds a desktop GUI using the native `tkinter` library.
    *   Outputs clean control string telemetry like `FORWARD: Speed 50`.
*   [ ] **Webcam Motion Detection Grid**
    *   Processes live laptop camera feeds using `OpenCV`.
    *   Alerts the terminal exactly which grid coordinate detected movement.
*   [ ] **State-Machine Path Simulator**
    *   Implements core Object-Oriented Programming (OOP) design patterns.
    *   Simulates robot logic states: `IDLE`, `SCANNING`, `MOVING`, `BLOCKED`.

### 🛡️ CYBERSECURITY AUTOMATION SCRIPTS

*   [ ] **Multi-Threaded Port Scanner**
    *   Utilizes the built-in `socket` library for network discovery.
    *   Identifies open ports and banners on local target IPs.
*   [ ] **Automated RegEx Log Parser**
    *   Scans large server logs using regular expressions (`re`).
    *   Flags malicious traffic patterns like SQL injection attempts.
*   [ ] **Ransomware Canary Monitor**
    *   Watches highly sensitive local folders using `watchdog`.
    *   Triggers system alerts if instant mass-file renaming occurs.
*   [ ] **SSH Compliance Auditor**
    *   Deploys automation across remote servers using `Paramiko`.
    *   Verifies system configurations against security baselines simultaneously.
*   [ ] **PCAP Packet Sniffer**
    *   Intercepts local wireless network traffic using `Scapy`.
    *   Decodes unencrypted packet headers to flag insecure protocols.
